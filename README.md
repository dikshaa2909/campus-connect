# 🎓 Campus Connect

Campus Connect is a full-featured College Management System built with **React**, **Tailwind CSS**, and **Firebase**. It provides an integrated portal for **students**, **faculty**, and **administrators** to manage academic operations, communication, and resources seamlessly.

---

## 🚀 Features

### 🧑‍🎓 Student Portal
- Secure Login/Signup
- View timetable, grades, and attendance
- Submit assignments
- View announcements and event schedules

### 👨‍🏫 Faculty Portal
- Manage and upload assignments
- Mark attendance
- Announce events, lectures, or deadlines
- View student data

### 🛠️ Admin Portal
- Manage users (students/faculty)
- Post global announcements
- Maintain master data (courses, subjects, etc.)
- Monitor system activity

### 🖥️ Common
- Firebase Authentication for secure login
- Real-time database using Firebase Firestore
- Responsive UI with Tailwind CSS
- Role-based access and routing

---

## 🧱 Tech Stack

| Technology | Description |
|------------|-------------|
| **React** | Frontend library for building dynamic UI |
| **Firebase** | Backend-as-a-Service for Authentication and Database |
| **Tailwind CSS** | Utility-first CSS framework for fast styling |
| **React Router** | Client-side routing for navigation |
| **Vite** | Fast build tool for React development |
